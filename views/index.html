Digiverse API

User Routes

GET / Shows all available end-points for this API
GET /users Shows all users on the database
POST /users Adds a new user to the database
GET /users/:id Shows the user with the specific ID on the database
PUT /users/:id Edits the user with the specific ID on the database
DELETE /users/:id Removes the user with the specific ID on the database

Product Routes

GET /products Shows all products on the database
POST /products Adds a new product to the database
GET /products/:id Shows the product with the specific ID on the database
PUT /products/:id Edits the product with the specific ID on the database
DELETE /products/:id DELETE the user with the specific ID on the database
