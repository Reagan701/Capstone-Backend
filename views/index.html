Digiverse API

User Routes

GET /users Shows all users on the database
POST /users Adds a new user to the database
GET /users/:id Shows the user with the specific ID on the database
PUT /users/:id Edits the user with the specific ID on the database
DELETE /users/:id Deletes the user with the specific ID from database

Product Routes

GET /products Shows all products on the database
POST /products Adds a new product to the database
GET /products/:id Shows the product with the specific ID on the database
PUT /products/:id Edits the product with the specific ID on the database
DELETE /products/:id Deletes the user with the specific ID from the database

Billing Routes

GET /billing Shows all billing information on the database
GET /billing/:id Shows all billing information of the user with the specific ID on the database
PUT /billing/:id Updates the billing information of the user with the specific ID
DELETE /billing/:id Clears the billing information of the user with the specific ID

Cart Routes

GET /carts Shows all cart information on the database
GET /cart/:id Shows all cart information of the user with the specific ID on the database
POST /cart/:id Updates the cart information of the user with the specific ID
DELETE /cart/:id Clears the cart of the user with the specific ID on the database
